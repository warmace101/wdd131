Site Plan - WSF Clan

Site Name

The site name "WSF Clan" was selected because it accurately represents the group, which is to provide advice, insight, and friends for gaming enthusiasts in the virtual world.

Site Purpose

The site serves as a central hub for hiking enthusiasts in the Bay Area, providing information on trails, hiking events, gear recommendations, and a platform for community engagement.

Scenarios

  • What is happening this season?
  • How can I get a higher guardian rank?
  • What weapons are good?
  • What is the meta for weapons?

Color Schema

The color schema will consist of the following:

  • Primary Color: #2E8B57 (used for headings, buttons)
  • Secondary Color: #FFD700 (used for accents, highlights)

Typography

The following fonts will be used:

  • Heading Font: 'Roboto', sans-serif
  • Body Font: 'Open Sans', sans-serif
